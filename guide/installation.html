# Cài Đặt

# Yêu Cầu Máy Chủ

Trước khi cài đặt FleetCart, hãy đảm bảo máy chủ của bạn đáp ứng các yêu cầu sau:

  • PHP 7.3+
  • MySQL 5.6+ hoặc MariaDB 10.0+
  • Tiện Ích Mở Rộng PHP Intl
  • Tiện Ích Mở Rộng PHP OpenSSL
  • Tiện Ích Mở Rộng PHP PDO
  • Tiện Ích Mở Rộng PHP Mbstring
  • Tiện Ích Mở Rộng PHP Tokenizer
  • Tiện Ích Mở Rộng PHP XML
  • Tiện Ích Mở Rộng PHP Ctype
  • Tiện Ích Mở Rộng PHP JSON

# Cài Đặt FleetCart

# Bước 1: Giải Nén Và Tải Lên

Giải nén gói tài liệu đã tải xuống. Đổi tên thư mục FleetCart thành tên thư mục mà bạn mong muốn và tải thư mục đó lên máy chủ web của bạn qua FTP hoặc Bảng Điều Khiển.

# Bước 2: Tạo Cơ Sở Dữ Liệu

Tạo một cơ sở dữ liệu cho FleetCart thông qua bảng điều khiển của máy chủ của bạn. Nếu máy chủ của bạn có phpMyAdmin, bạn cũng có thể tạo cơ sở dữ liệu bằng phpMyAdmin.

# Bước 3: Chạy Trình Hướng Dẫn Cài Đặt

Truy cập địa chỉ trang web của bạn, sau đó bạn sẽ thấy trình hướng dẫn cài đặt.

CẢNH BÁO

Bạn nên cấu hình tài liệu / thư mục web gốc của máy chủ web của bạn là thư mục public.

# Tiền Cài Đặt

Trang Tiền Cài Đặt kiểm tra xem máy chủ của bạn có đáp ứng yêu cầu và có quyền truy cập đúng để thiết lập FleetCart hay không.

Các thư mục trong thư mục storage và bootstrap/cache phải có quyền ghi bởi máy chủ web của bạn, nếu không FleetCart sẽ không hoạt động.

# Cấu Hình

Bạn cần điền thông tin kết nối cơ sở dữ liệu, thông tin quản trị, và chi tiết cửa hàng, sau đó nhấp vào nút Cài Đặt để cài đặt FleetCart.

# Hoàn Thành

Bạn sẽ thấy thông báo thành công sau khi cài đặt thành công FleetCart. Bạn bây giờ có thể duyệt cửa hàng trực tuyến hoặc đăng nhập vào bảng điều khiển quản trị.
